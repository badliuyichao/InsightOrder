# 图表点击穿透功能 - 测试指南

## 功能说明

本次更新为 InsightOrder 工单分析系统添加了**图表点击穿透功能**，支持在所有17个图表上点击查看详细的工单列表。

## 已实现的功能

### ✅ 核心功能

1. **图表点击事件** - 所有17个图表类型都已支持点击事件
2. **工单详情对话框** - 弹出式对话框展示筛选后的工单列表
3. **数据筛选** - 根据点击的图表元素自动筛选相关工单
4. **搜索功能** - 支持搜索关键字、概要、经办人、项目名称
5. **状态筛选** - 支持按状态筛选工单（待分析、研发已完成、支持确认完成）
6. **分页展示** - 每页20条，支持调整每页显示数量（10/20/50/100）
7. **表格排序** - 所有列都支持排序功能

## 测试步骤

### 1. 启动项目

```bash
npm run dev
```

访问 http://localhost:3000

### 2. 加载数据

- 点击"选择 Excel 文件"按钮
- 选择 `orders/` 文件夹中的 Excel 文件
- 等待数据加载完成

### 3. 测试图表点击

#### 测试折线图（工单趋势分析）
1. 找到"工单趋势分析"图表
2. 鼠标悬停在数据点上，应显示手型指针
3. 点击任意数据点
4. 应弹出工单详情对话框
5. 对话框标题显示"工单详情列表"
6. 筛选条件显示"创建月份 = 2026-XX"
7. 表格显示该月所有工单

#### 测试柱状图（经办人工作量分布）
1. 找到"经办人工作量分布"图表
2. 点击任意柱子
3. 对话框显示该经办人的所有工单
4. 筛选条件显示"经办人 = XX"

#### 测试饼图（问题类型分布）
1. 找到"问题类型分布"图表
2. 点击任意扇区
3. 对话框显示该问题类型的所有工单
4. 筛选条件显示"客户问题类型 = XX"

#### 测试雷达图（经办人效率分析）
1. 找到"经办人效率分析（Top 5）"图表
2. 点击任意数据点
3. 对话框显示该经办人的所有工单

#### 测试漏斗图（状态转化漏斗）
1. 找到"状态转化漏斗"图表
2. 点击任意层级
3. 对话框显示该状态的所有工单

#### 测试散点图（超期风险识别）
1. 找到"超期风险识别"图表
2. 点击任意散点
3. 对话框显示该具体工单

### 4. 测试对话框功能

#### 搜索功能
1. 打开任意图表的详情对话框
2. 在搜索框输入关键字（如工单关键字、经办人姓名等）
3. 表格应实时筛选显示匹配的工单
4. 清空搜索框，表格恢复显示所有工单

#### 状态筛选
1. 打开详情对话框
2. 点击"状态筛选"下拉框
3. 选择"待分析"、"研发已完成"或"支持确认完成"
4. 表格只显示对应状态的工单

#### 分页功能
1. 打开详情对话框（确保工单数量 > 20）
2. 查看底部分页器，显示总条数和当前页码
3. 点击"下一页"或页码数字切换页码
4. 调整"每页显示条数"（10/20/50/100）
5. 表格相应调整显示条数

#### 表格排序
1. 点击任意列标题（经办人、状态、创建时间、处理时长、紧急程度）
2. 表格按该列升序/降序排序
3. 再次点击切换排序方向

#### 关闭对话框
- 点击右上角"×"按钮
- 点击对话框外部遮罩层
- 按 Esc 键

## 技术实现

### 核心文件

1. **useChartClick.ts** - 图表点击逻辑处理
   - 位置：`src/composables/useChartClick.ts`
   - 功能：构建筛选条件、过滤工单数据

2. **DetailDialog.vue** - 工单详情对话框组件
   - 位置：`src/components/common/DetailDialog.vue`
   - 功能：展示工单列表、搜索、筛选、分页

3. **图表组件扩展**
   - BaseChart.vue
   - RadarChart.vue
   - FunnelChart.vue
   - SankeyChart.vue
   - 添加了点击事件监听和 emit

4. **Dashboard.vue** - 主仪表盘
   - 集成了所有图表的点击事件处理
   - 管理详情对话框状态

### 数据流程

```
用户点击图表元素
    ↓
图表组件 emit('chartClick', params)
    ↓
Dashboard.vue handleChartClick()
    ↓
useChartClick.buildFilterCondition()
    ↓
useChartClick.filterTickets()
    ↓
打开 DetailDialog 显示工单列表
```

## 注意事项

### 1. 热力图和桑基图
- 热力图需要双维度筛选，点击逻辑较复杂
- 桑基图区分节点和连线点击，分别处理

### 2. 性能考虑
- 使用 computed 缓存筛选结果
- 分页避免一次性渲染大量数据
- 大数据量时建议使用虚拟滚动（可选优化）

### 3. 用户体验
- 图表鼠标指针已设置为手型
- 对话框支持 Esc 键关闭
- 点击遮罩层可关闭对话框
- 表格支持列排序和宽度调整

## 已知问题和限制

1. **热力图元数据** - 当前热力图的 chartMeta 信息可能需要进一步优化，以确保双维度筛选准确
2. **桑基图复杂路径** - 桑基图连线点击的筛选逻辑可能需要根据实际数据结构调整
3. **大数据量性能** - 当筛选结果超过1000条时，表格渲染可能有性能问题（可通过虚拟滚动优化）

## 后续优化建议

1. **导出 Excel 功能** - 添加导出按钮，将筛选结果导出为 Excel 文件
2. **批量操作** - 支持勾选多个工单进行批量操作
3. **高级筛选** - 支持多条件组合筛选
4. **虚拟滚动** - 使用 el-table-v2 处理大数据量
5. **图表联动** - 点击一个图表后，其他图表联动筛选显示
