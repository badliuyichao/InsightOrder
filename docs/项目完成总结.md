## 🚀 如何使用

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

访问：http://localhost:3000

### 3. 使用流程
1. 点击右上角"选择文件"按钮
2. 选择 Excel 工单文件（.xls 或 .xlsx）
3. 系统自动解析并展示所有分析图表
4. 点击"刷新数据"按钮可更新数据

### 4. 生产构建
```bash
npm run build
```

### 5. 预览生产版本
```bash
npm run preview
```
